AUFGABENSTELLUNG
 Laden Sie aktuelle Eclipse Java EE Version herunter!
 Extrahieren Sie Eclipse und starten Sie Eclipse.
 Wählen Sie einen beliebigen Ordner als Ihren neuen Workspace aus.
 Erstellen Sie ein neues Eclipse Projekt!
 Benennen Sie Ihr Projekt nach folgendem Schema:
uebung01_USERNAME
 Erstellen Sie ein neues Package at.fhj.swd.demo
 Erstellen Sie eine neue Java Klasse User.java
 Erstellen Sie einen leeren Konstruktur mit keinen Übergabeparametern.
 Legen Sie in dieser Klasse eine neue Variable „username“ an (Datentyp String), diese darf
nur innerhalb ihrer Java Klasse verwendbar/zugreifbar sein.
 Erstellen Sie für die Variable „username“ eine get- sowie eine set-Methode.
 Erstellen Sie ein Verzeichnis (docs) für Dokumente und diversen anderen Dateien.
 Erstellen Sie im Verzeichnis docs eine „ReadMe.txt“ Datei und beschreiben Sie darin
kurz Ihr Java Projekt.
 Exportieren Sie Ihr Eclipse Java Projekt mit der Exportfunktion! Benennen Sie Ihr ZipArchiv
nach folgendem Schema:
uebung01_USERNAME.zip
 Geben Sie ihr exportiertes Projekt in Moodle ab!
